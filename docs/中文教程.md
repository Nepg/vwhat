# HumanitZ 无头专用服务器 – 物品堆叠模组自定义教程

## 1. 工具准备

需要以下两个工具，请自行下载并放置到 `Customization` 目录（与 `build.bat`、`blacklist.txt` 同级）：

- **UAssetGUI**（UE4 资产编辑器）
  下载：[Releases · atenfyr/UAssetGUI · GitHub](https://link.wtturl.cn/?target=https%3A%2F%2Fgithub.com%2Fatenfyr%2FUAssetGUI&scene=im&aid=497858&lang=zh) 文件：`UAssetGUI.exe`
- **repak**（PAK 打包工具）
  下载：[Releases · trumank/repak · GitHub](https://link.wtturl.cn/?target=https%3A%2F%2Fgithub.com%2Ftrumank%2Frepak&scene=im&aid=497858&lang=zh) 文件：`repak.exe`

放置路径示例：

```
Customization/
├── build.bat
├── blacklist.txt
├── UAssetGUI.exe
├── repak.exe
└── ...
```

## 2. 配置物品黑名单

列入黑名单的物品堆叠上限将设置为 **1**（不可堆叠）；未列入的物品堆叠上限将设置为 **999**。

编辑 `blacklist.txt`，规则如下：

- 每行一个物品的 **ID**（如 `Apple`，如果你不确定物品id，可以考虑安装HumanitZ的目录——它列出了700多个物品id供参考。
- 以 `#` 开头的行视为注释，忽略处理。

修改后保存文件。

## 3. 使用 UAssetGUI 修改数据库资产

1. 运行 `UAssetGUI.exe`，在右上角引擎版本下拉菜单中选择 **UE4.27**。
2. 打开数据库文件： `File` → `Open` → 选择 `Customization\ZzzHlSvrStack_p\HumanitZServer\Content\TSS_Game\Data\Localization\DT_ItemDatabase.uasset`
3. 启用脚本权限：`Edit` → `Settings` → 勾选 **Allow untrusted scripts**，关闭设置。
4. 创建脚本：`Utils` → `Edit scripts` → `Add new script`，新建空白脚本。
5. 将 Customization 内的 `modify_stack.cs` 的全部代码粘贴到脚本编辑器中，保存。
6. 执行脚本：`Utils` → `Execute script` → 选择刚创建的脚本，运行。
7. 脚本执行完毕后会弹出提示框，显示修改的物品总数，确认后点击 `OK`。
8. 保存资产：`File` → `Save`，然后关闭 UAssetGUI。

## 4. 打包生成服务器模组

确保 `repak.exe` 已位于 `Customization` 目录，双击运行 `build.bat`。
打包完成后，目录下会生成 `ZzzHlSvrStack_p.pak`，此即为服务器可加载的模组文件。😸
